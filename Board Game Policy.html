Policies and Rules governing Huie Library's new board game collection.

Board Game Circulation Policy

  1. Board games have a three day loan period with the ability to renew the item for an additional three days.
  2. Patrons can check out up to three games or puzzles at one time.
  3. Games must be returned inside the library. Please do not place them in the book drops.
  4. The board game is not considered returned until all items within the box/case have been returned to the library in working and complete condition. A lost, damaged or stolen game will result in a minimum charge of $10 up to the full replacement charge cost.
  5. This is a loan program ONLY. The library staff is not available for instruction. Detailed instructions are provided in the game box.
